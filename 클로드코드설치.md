# 새 Windows 11 환경 Claude Code 작업환경 재구축 가이드

Windows 11 native (WSL 없음) + VS Code 터미널(PowerShell) 환경에서
Claude Code + MCP 생태계를 처음부터 구축하는 순서.

## 현재 구성 스냅샷 (옮겨야 할 것)

- **MCP 서버**: `serena`, `sequential-thinking`
- **플러그인**:
  - `code-review@claude-plugins-official`
  - `figma@claude-plugins-official`
  - `discord@claude-plugins-official`
  - `codex@openai-codex`
- **마켓플레이스**: `anthropics/claude-plugins-official`, `openai/codex-plugin-cc`
- **전역 설정**: `%USERPROFILE%\.claude\CLAUDE.md`, `settings.json`, `settings.local.json`, `commands\`
- **MCP 설정**: `%USERPROFILE%\.claude.json`
- **auto-memory**: `%USERPROFILE%\.claude\projects\<프로젝트-해시>\memory\`

> macOS 경로 `~/.claude/` ↔ Windows 경로 `%USERPROFILE%\.claude\`
> (예: `C:\Users\<YourName>\.claude\`)

---

## 0. 백업 (기존 macOS 머신에서 먼저 뽑기)

```
~/.claude.json
~/.claude/CLAUDE.md
~/.claude/settings.json
~/.claude/settings.local.json
~/.claude/commands/
~/.claude/projects/<hash>/memory/
```
Windows 로 옮긴 뒤 `%USERPROFILE%` 아래 동일 구조로 복사.

---

## 1. 시스템 툴체인 (PowerShell 관리자)

Windows 기본 패키지 매니저는 `winget`.

```powershell
# Git
winget install --id Git.Git -e

# Node.js LTS (Claude Code CLI + npx 기반 MCP 서버용)
winget install --id OpenJS.NodeJS.LTS -e

# uv (serena 실행용 Python 런타임)
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```

설치 후 **새 PowerShell 창** 을 열어 PATH 반영 확인:
```powershell
git --version
node --version
npm --version
uv --version
```

---

## 2. Claude Code CLI 설치 + 로그인

```powershell
npm install -g @anthropic-ai/claude-code
claude      # 최초 실행 시 브라우저 OAuth 로그인
```
로그인하면 `%USERPROFILE%\.claude\` 가 자동 생성됨.

---

## 3. 전역 설정 복원

백업한 파일을 아래 위치로 복사:

| 백업 원본 | Windows 대상 |
|---|---|
| `~/.claude.json` | `%USERPROFILE%\.claude.json` |
| `~/.claude/CLAUDE.md` | `%USERPROFILE%\.claude\CLAUDE.md` |
| `~/.claude/settings.json` | `%USERPROFILE%\.claude\settings.json` |
| `~/.claude/settings.local.json` | `%USERPROFILE%\.claude\settings.local.json` |
| `~/.claude/commands/` | `%USERPROFILE%\.claude\commands\` |
| `~/.claude/projects/<hash>/memory/` | `%USERPROFILE%\.claude\projects\<hash>\memory\` |

**중요**: 복사한 `.claude.json` 의 `mcpServers.serena` 블록은 macOS 절대경로
(`/Users/leejae/...`) 라서 Windows 경로로 반드시 교체해야 동작함. (4번 참고)

## 4. MCP 서버 등록

### sequential-thinking
```powershell
claude mcp add sequential-thinking -s user -- npx -y @modelcontextprotocol/server-sequential-thinking
```

### serena
```powershell
# 작업 루트를 먼저 결정 (예: C:\Dev\chalk_4_game)
cd C:\Dev\chalk_4_game
git clone https://github.com/oraios/serena.git

claude mcp add serena -s user -- uv run --directory C:\Dev\chalk_4_game\serena serena-mcp-server --context=claude-code --project C:\Dev\chalk_4_game
```

> **교훈**
> - `--project-from-cwd` 는 serena 자기 디렉토리를 프로젝트로 잡는 버그가 있음 → `--project` 명시.
> - MCP 설정을 바꿨으면 `/mcp` 재연결로는 반영 안 됨. Claude Code 세션을 `/exit` 하고 다시 `claude` 실행할 것.

등록 확인:
```powershell
claude mcp list
```

## 5. 플러그인 마켓플레이스 + 플러그인 설치

Claude Code 세션 안에서 순서대로 입력:
```
/plugin marketplace add anthropics/claude-plugins-official
/plugin marketplace add openai/codex-plugin-cc

/plugin install code-review@claude-plugins-official
/plugin install figma@claude-plugins-official
/plugin install discord@claude-plugins-official
/plugin install codex@openai-codex
```
각 플러그인의 skill / 추가 MCP 는 자동 등록됨.

### 플러그인별 초기 세팅
- **discord**: `/discord:configure` 로 봇 토큰 입력 → `/discord:access` 로 페어링
- **figma**: Figma Desktop 설치 → Preferences 에서 "Enable local MCP Server" 체크
- **codex**: `/codex:setup` 으로 Codex CLI 상태 확인 (필요 시 `winget install OpenAI.Codex`)

## 6. VS Code 연동

1. Extensions → "Claude Code" (Anthropic 공식) 설치
2. VS Code 터미널 기본 프로필을 **PowerShell** 로 유지
   (Settings → `terminal.integrated.defaultProfile.windows` = `PowerShell`)
3. 터미널에서 `claude` 실행 → IDE 자동 연결
4. `Ctrl+Esc` 로 Claude Code 패널 토글 (Windows 단축키)

---

## 7. Unity 개발 환경

1. Unity Hub for Windows 다운로드 (unity.com/download)
2. 프로젝트가 요구하는 Unity Editor 버전 설치
3. 프로젝트 오픈 시 매니페스트에 Unity MCP 패키지가 있으면 자동 활성화
4. Unity 에디터가 켜져 있을 때만 `mcp__UnityMCP__*` 툴이 Claude Code 에 노출됨

## Windows 특유의 자주 막히는 지점

- **`.claude.json` 의 macOS 경로**: `/Users/leejae/...` 가 그대로면 serena 가 연결 실패. Windows 경로로 치환.
- **JSON 안 백슬래시**: `.claude.json` 에 경로를 직접 적을 땐 `C:\\Dev\\chalk_4_game` 처럼 이스케이프.
- **PATH 미반영**: winget / uv 설치 직후 기존 터미널은 PATH 가 구식. **새 PowerShell 창**을 띄워야 반영됨.
- **PowerShell 실행 정책**: uv install 스크립트가 막히면
  `Set-ExecutionPolicy -Scope CurrentUser RemoteSigned`.
- **npm 전역 설치 권한**: `EACCES` / 권한 에러 → 관리자 PowerShell 에서 재실행.
- **Git Long Path**: clone 시 `Filename too long` → 관리자 PowerShell 에서
  `git config --system core.longpaths true`.
- **Claude Code MCP 재연결 미반영**: 설정 변경 후 `/mcp` 재연결 X, 반드시 `/exit` 후 `claude` 재실행.
